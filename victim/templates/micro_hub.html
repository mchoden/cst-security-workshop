🏗️ Microservices Security

An API gateway sits at /microservices/api/gateway/... and proxies to internal services. Find the misconfigurations.

Level 1

No Service Auth

The gateway proxies to the user service. Does the internal service require auth?

Medium
Hint

SSRF through the URL preview to bypass the gateway: http://localhost:5002/user-service/users

Level 2

Debug Endpoint

A debug endpoint was left enabled in the internal service.

Medium
Hint

Try http://localhost:5002/debug via SSRF or the gateway

Level 3

Gateway Auth Bypass

The API gateway checks auth on GET but forgot other methods...

Hard
Hint

curl -X POST http://localhost:5000/microservices/api/gateway/payment-service/process -H "Content-Type: application/json" -d '{"amount":100}'
